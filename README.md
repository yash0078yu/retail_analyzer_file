# Retail Sales Analyzer

## Objective
This project is used to analyze retail sales data using Python.  
It demonstrates Object-Oriented Programming, NumPy, Pandas, and data visualization techniques.

## Files Included
- retail_analyzer.py  
- retail_sales.csv  
- README.md  

## Dataset Description
The dataset contains the following columns:
- Date  
- Product  
- Category  
- Price  
- Quantity Sold  
- Total Sales  

## Features
- Load data from a CSV file  
- Clean and validate data  
- Calculate sales metrics  
- Perform category-wise and date-wise analysis  
- Filter data by category and date range  
- Visualize data using charts  

## Technologies Used
- Python  
- NumPy  
- Pandas  
- Matplotlib  
- Seaborn  

## Output
- Sales metrics displayed in the terminal  
- Bar chart for category sales  
- Line graph for sales trend  
- Heatmap for correlation  

## Conclusion
This project fulfills practical exam requirements and demonstrates retail data analysis using Python.
