# retail_analyzer_file

Objective
This project is used to analyze retail sales data using Python.
It demonstrates the use of Object-Oriented Programming, NumPy, Pandas, and data visualization libraries.
Files Included
retail_analyzer.py – Main Python program
retail_sales.csv – Dataset file
README.md – Project description
Dataset Description
The CSV file contains the following columns:
Date
Product
Category
Price
Quantity Sold
Total Sales
Features
Load data from CSV file
Clean and validate data
Calculate sales metrics
Category-wise and date-wise analysis
Filter data by category and date range
Visualize data using charts
Technologies Used
Python
NumPy
Pandas
Matplotlib
Seaborn
